## Types of Headings:
h1 - Main Headings

h2 - Subheadings
 
h3, h4, h5, h6 - Different levels of subheadings

p - Paragraph

## Syntax:

Comments: 

`<--! text -->`

img - element used to add images

alt - attribute for img element incase the image does not load this text is displayed. This also helps increase accessibility to screen readers.

Format:
`<img src="https://www.freecatphotoapp.com/your-image.jpg" alt="A business cat wearing a necktie.">`

src - source of an element

Hyperlink:

Anchor element (a)

<a href = "link"> text to be hyperlinked </a>

Internal Hyperlink:

`<a href = "#id"> Text to be hyperlinked </a> `

Open in New Tab: 

`target="_blank"`

Dead Link:

```HTML
target="#"
```

Unordered List:
```HTML
<ul> 
  <li> Item 1 </li>
  <li> Item 2 </li> 
</ul> 
```

Ordered List:
```HTML
<ol>
  <li> Item 1</li>
  <li> Item 2</li>
</ol>
```

Input Elements (Text Type):
``` HTML
<input type = "text"> 
```

Placeholder For Input:
```HTML
<input type = "text" placeholder = "This is placeholder Text">
```

Submit Input Data:
```HTML
<form action = "/url-where-you-want-to-submit-form-data" >
  <input>
</form>
```

Submit Button:
```HTML
<button type = "submit"> this button submits the form </button>
```
